03. Engineering Case Studies

Creator Intelligence

A B2B SaaS platform for AI-powered YouTube comment moderation. Automatically classifies comments as toxic, spam, or safe using Groq (llama-3.3-70b) with confidence-based auto-moderation thresholds (≥0.9 auto-hide, 0.7–0.9 flag for review).

Engineering Challenge: Building a fully event-driven pipeline — a cron job fetches comments from YouTube Data API v3, pushes to a BullMQ queue, and a worker classifies and auto-hides high-confidence toxic comments via YouTube API. Analytics API with Redis caching tracks per-channel stats, per-video toxicity breakdown, and 7-day sentiment trends via daily SentimentSnapshot aggregation.

Customer Agent

An AI-powered customer support backend that automatically classifies tickets, routes them using intent detection, and drafts suggested replies using a RAG pipeline — combining a ChromaDB vector store with HuggingFace embeddings for knowledge base search.
