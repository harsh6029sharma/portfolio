SocialHive Microservices Platform

A robust backend for a social media platform built entirely on a microservices architecture. It features a centralized API Gateway routing to distinct Identity, Post, Media, and Search services.

ImageVault API

A secure, role-based REST API for permission-based image storage. Integrates MongoDB for persistence and Cloudinary for robust cloud-based image hosting.

Problem Solved: Ensuring secure file uploads and strict access control. Admins have upload/delete privileges with ownership checks to prevent data loss, while standard users have read-only access, protected via JWT.
